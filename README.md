# IPL Analysis Dashboard (2008–2025)

## Project Overview
This project presents an interactive IPL (Indian Premier League) analysis dashboard built using Power BI. It covers match data from 2008 to 2025, providing insights into team performance, player statistics, and match trends.

## Features
- Season-wise analysis  
- Winning teams and performance trends  
- Top players (runs, wickets, strike rate)  
- Venue-based insights  
- Interactive visualizations and filters  

## Tools and Technologies
- Power BI – Data visualization  
- DAX (Data Analysis Expressions) – Measures and calculations  
- Excel / Dataset – Data source
